🎯 雅思学习面板 · 使用说明
================================

📦 里面有什么
────────────────
- index.html       👉 主入口(双击打开这个)
- speaking.html    口语范例答案(Part 1/2/3,共 599 题)
- listening.html   听力考词听写练习(747 词)
- audio/           听力单词的英音朗读文件(勿删)

🖥️ 怎么使用
────────────────
1. 把整个文件夹解压到你喜欢的位置(桌面、文档夹都行)
2. 打开 site 文件夹
3. 双击 index.html
4. 用浏览器打开(推荐 Chrome / Edge / Safari)
5. 点卡片进入 口语 或 听力,尽情使用!

⚠️ 重要提示
────────────────
📌 保持 audio 文件夹和 listening.html 在同一目录里,否则听力页没声音
📌 建议在电脑上使用,手机浏览器体验较差
📌 用微信/QQ 内置浏览器打开可能有问题,请用系统浏览器
📌 学习进度保存在你自己的浏览器里,清缓存/换设备会重置

💡 快捷键 & 小贴士
────────────────
· 听力页: Tab 键重听发音,Enter 键检查/下一个
· 听力页: 勾选「翻译」显示中文,可区分同音词
· 听力页: 完成后能只练错词、按章节看掌握度
· 口语页: 点答案可编辑,自动保存;可按学习状态过滤
· 两个页面顶部都有「← 面板」按钮,可回到入口

🔊 关于发音
────────────────
· 听力页 747 个单词的发音已经打包在 audio 文件夹里(英式英语)
· 双击点击喇叭图标即可听发音,不用联网、任何电脑都是一致效果
· 「语速」下拉可调整快慢

祝备考顺利!🌟
