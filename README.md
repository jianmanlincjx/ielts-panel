# 🎯 雅思学习面板

自制的雅思备考小工具，汇总 2026 年 6-8 月大陆题库。

## 内容

- 🗣️ **口语范例答案** — Part 1/2/3 共 599 题，含中文提示、句型 tip、可编辑答案
- 🎧 **听力考词听写** — 747 个答案词，交互式听写练习
  - 点喇叭听发音 → 打字判分 → 中文提示防同音混淆
  - 熟练度自动记录（Leitner 5 盒记忆法）
  - 按主题分章节练习，错词一键复练
  - 内置 638 个单词英音朗读 MP3（en-GB Ryan Neural）

## 使用

- **在线**：打开 [首页链接](./index.html)（部署到 Pages 后会有公开链接）
- **本地**：下载后双击 `index.html`

学习进度保存在浏览器 localStorage，同一浏览器打开就在，清缓存会重置。

## 技术

单一静态网站，纯 HTML + CSS + JS，无框架无后端。发音使用 Microsoft Edge TTS 预生成的 MP3，兼容所有浏览器。
